# How to Evaluate AI Tools: A Guide for Teachers

## Goal
New AI tools for educators appear every day. This guide provides a simple, structured framework to help you evaluate which tools are worth your time, ensure they protect student privacy, and verify their pedagogical value before bringing them into your classroom.

### 🛠️ The "3-P" Evaluation Framework
*   **Privacy:** Does the tool comply with GDPR and local school data laws?
*   **Pedagogy:** Does it actually enhance learning, or just automate a superficial task?
*   **Practicality:** Is it affordable, accessible, and easy for students to use?

---

## 🔄 The Evaluation Workflow

`[Initial Discovery] ➔ [Security Screen] ➔ [Pedagogical Test] ➔ [Accessibility Check] ➔ [Pilot]`

### Step 1: Security & Privacy Screen
Before looking at features, check the "Privacy Policy."
*   Does it require student accounts?
*   Does it sell data to 3rd parties?
*   Is it an "Education" version with higher protections?

### Step 2: Pedagogical Value Test
Ask yourself: "If I use this tool, what skill is the student *not* practicing?"
*   **Good use:** AI automates a tedious task to free up time for high-level analysis.
*   **Bad use:** AI performs the exact task the student is supposed to be learning (e.g., solving a basic math problem without showing steps).

### Step 3: Accessibility Check
*   Does the free version offer enough value?
*   Does it work on all devices (mobile, tablet, Chromebook)?
*   Does it have built-in accessibility features (text-to-speech, high contrast)?

---

## ✍️ Effective Prompting for Tool Evaluation

### Prompt Template:
> "Act as an Educational Technology auditor. I am considering using [Tool Name] for my [Subject] class. Based on its official website and public reviews, analyze its potential for [Specific Task]. Highlight potential privacy concerns and suggest how it could be used to support [Learning Objective]."

### Example Prompt:
> "Act as an EdTech auditor. I am considering using 'Canva Magic Media' for my 8th-grade Geography class to generate images of different biomes. Analyze its pros and cons for this age group, focusing on safety filters and how it might spark student creativity."

---

## ⚠️ The "Human-in-the-Loop" Review Checklist
*   **Terms of Service:** **MANDATORY:** Check the minimum age requirement. Many AI tools are 13+ or 18+.
*   **Bias Check:** Does the tool generate stereotypical or biased content? (e.g., only showing certain demographics in professional roles).
*   **The "Hype" Filter:** Don't be swayed by marketing. Test the tool with a difficult example to see its true limitations.
