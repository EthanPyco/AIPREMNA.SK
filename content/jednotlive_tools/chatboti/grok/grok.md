# Grok: Standardized Guide for Teachers

---

## 1. General Use Case
Grok is an AI developed by xAI, integrated directly into the X (formerly Twitter) platform. Its unique strength is real-time access to global conversations and a "Fun Mode" that allows for a more witty, rebellious personality. For teachers, it is best used for analyzing current events, debating modern topics, and gathering up-to-the-minute information.

### 🛠️ Recommended Use Cases
* **[Summarizing Notes](../../../jednotlive_usecases/sumarizacia_uciva/sumarizacia_poznamok/summarizing_notes.md):** Breaking down real-time news threads or trending educational discussions into concise summaries.
* **[Writing Emails](../../../jednotlive_usecases/administrativa/pisanie_emailu/writing_email.md):** Using the "Regular" mode to draft professional correspondence with high factual awareness.
* **[Practice Questions](../../../jednotlive_usecases/aktivity_na_hodinu/cvicne_ulohy/practice_questions.md):** Generating debate prompts based on current world events or recent scientific discoveries.

## 🔄 The Step-by-Step UI Workflow

`[Access/Login] ➔ [Prompting/Input] ➔ [Interacting/Refining] ➔ [Finalizing/Exporting]`

### Step 1: Accessing the Tool
Grok is primarily accessed through x.com (Premium subscription required). Click the Grok icon in the sidebar. It is also available via the X mobile app.

### Step 2: Providing Input
Choose between "Fun Mode" (for creative brainstorming) and "Regular Mode" (for standard school tasks). Enter your prompt in the chat box. You can ask Grok to search for specific news from the last hour to provide context for your lesson.

### Step 3: Refining the Result
Review the live sources linked in Grok’s response. You can refine by asking for "more academic sources" or telling Grok to "remove the sarcasm" if you are in Fun Mode and need a more serious tone for a handout.

## ✍️ Effective Prompting for Grok

Grok thrives on real-time data and conversational depth.

### How to Write Prompts for this Tool:
* **Search Specificity:** Use prompts like "Search for the latest 2024 updates on [Topic] and summarize the consensus."
* **Tone Control:** Explicitly state if you want a "strictly professional" or "humorous student-centric" explanation.
* **Source Analysis:** Ask Grok to "compare the top three perspectives currently trending on X regarding this educational policy."

### Example Prompt:
> "Search for the latest news regarding NASA's Artemis mission from the last 48 hours. Create a 3-bullet point update that I can read to my 5th-grade science class. Keep the language exciting but scientifically accurate."

---

## ⚠️ Tool-Specific Review Checklist (Shortcomings)

* **Source Reliability:** Because Grok pulls from X, it may encounter "fake news" or biased opinions. Always verify claims against established news outlets or textbooks.
* **Fun Mode Risks:** Fun Mode can occasionally be too "edgy" for a primary school environment. Always preview the output before sharing with students.
* **Access Requirements:** Grok currently requires a paid X Premium subscription, which may not be available for all staff budgets.
* **Privacy Warning:** Do not share student names or private school data on the X platform.

## ✅ How to Know You Have Mastered This Tool
By the end of this guide, you should be able to:
* Successfully use Grok to find and summarize a real-time news event for a classroom discussion.
* Toggle effectively between Fun and Regular modes based on whether you are brainstorming or drafting.
* Verify Grok's claims by checking the provided X-thread citations.
