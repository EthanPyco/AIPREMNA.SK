## 🔄 The Step-by-Step UI Workflow

`[Access/Login] ➔ [Prompting/Input] ➔ [Interacting/Refining] ➔ [Finalizing/Exporting]`

### Step 1: Accessing the Tool
Navigate to chat.deepseek.com or use the DeepSeek mobile app. You can log in using an email address or via third-party accounts (like Google). The interface is streamlined for direct interaction without complex configuration.

### Step 2: Providing Input
Enter your request in the chat interface at the bottom of the screen. DeepSeek allows for long-form text input and file uploads (via the attachment icon). For complex logical tasks, you can select specific models like "DeepSeek-R1" to leverage its advanced reasoning chain.

### Step 3: Refining the Result
Review the output and provide follow-up instructions to narrow down the focus. DeepSeek is particularly good at following multi-step corrections. If the tone is too technical, ask it to "simplify for a younger audience" or "make the tone more encouraging."

## ✍️ Effective Prompting for DeepSeek

DeepSeek thrives on logical structure and clear objectives.

### How to Write Prompts for this Tool:
* **Encourage Reasoning:** For complex topics, start with "Think step-by-step" or use the R1 model to see the "thought process" behind the answer.
* **Structural Constraints:** Use specific formatting requests like "Return the output as a Markdown table" or "List the key concepts in bullet points followed by a summary."
* **Role Specification:** Clearly define the educational context (e.g., "You are a secondary school Physics teacher explaining thermodynamics").

### Example Prompt:
> "You are an expert curriculum designer. I want you to create a weekly lesson plan for a 7th-grade Biology unit on 'Plant Cells'. The output should include learning objectives, a list of required materials, and a 10-minute introductory activity. Style: Structured and easy to follow."

---

## ⚠️ Tool-Specific Review Checklist (Shortcomings)

* **Hallucination Check:** Like all LLMs, DeepSeek can occasionally generate incorrect facts. Verify technical data, historical dates, and specific citations against trusted educational resources.
* **Technical Bias:** DeepSeek can sometimes default to a very concise or technical tone. Teachers may need to prompt it specifically for "empathetic" or "warm" language when writing to parents.
* **Regional Knowledge:** While highly capable globally, ensure local curriculum standards are explicitly mentioned in your prompt to avoid generic "global" examples.
* **Privacy Warning:** Do not upload identifiable student information (names, ID numbers, or private grades) into the chat interface.
