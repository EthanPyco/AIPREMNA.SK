# Microsoft Copilot: Standardized Guide for Teachers

---

## 1. General Use Case
Microsoft Copilot is an AI assistant integrated into the Windows ecosystem and Microsoft 365. For teachers in "Microsoft schools," it serves as a powerful bridge between web searching and document creation, allowing you to generate content, analyze web pages, and create images all within a secure, familiar environment.

### 🛠️ Recommended Use Cases
* **Writing Emails:** Drafting messages within Outlook or the Copilot web interface.
* **Creating Exams:** Using "Notebook" mode to paste long curriculum standards and generate structured assessments.
* **Exam Grading:** Utilizing Copilot's strong OCR and data protection features to help analyze student work against rubrics.
* **Creating Cheat Sheets:** Combining web search results with your own Word documents to create summaries.

---

## 🔄 The Step-by-Step UI Workflow

`[Access/Login] ➔ [Prompting/Input] ➔ [Interacting/Refining] ➔ [Finalizing/Exporting]`

### Step 1: Accessing the Tool
Access Copilot via copilot.microsoft.com, the Edge browser sidebar, or the Copilot app. Log in with your school (M365) account to ensure "Commercial Data Protection" is active.

### Step 2: Providing Input
Choose a conversation style: **Creative** (best for brainstorming/images), **Balanced** (general tasks), or **Precise** (best for facts/math). Use the "Notebook" tab for long-form prompts (up to 18,000 characters) where you need to provide significant context.

### Step 3: Refining the Result
Copilot provides clickable citations for its claims. Click these to verify sources. You can also use the "Export" button to send the results directly to Word, PDF, or text format.

---

## ✍️ Effective Prompting for Microsoft Copilot

Copilot excels when you utilize its connection to the web and its different "styles."

### How to Write Prompts for this Tool:
* **Set the Style First:** Use "Precise" mode for science and math questions to reduce errors. Use "Creative" for writing poems or stories.
* **Verify via Citations:** Always check the footnotes provided by Copilot to ensure the information comes from a reputable educational source.
* **Use Notebook for Complexity:** If you have a 5-page lesson plan to summarize, use the "Notebook" mode for better focus.

### Example Prompt:
> "Using the 'Precise' style, research the current 2024 educational standards for Algebra 1 in Slovakia. Compare these with the provided syllabus [Paste Text]. Identify any missing topics and suggest three practice problems for each."

---

## ⚠️ Tool-Specific Review Checklist (Shortcomings)

* **Source Reliability:** Copilot pulls from the live web. Occasionally, it may cite blogs or unreliable forums. Always check the linked sources.
* **Formatting:** When exporting to Word, check for layout shifts or font changes that may occur during the transfer.
* **Protection Check:** Look for the "Protected" icon near your profile to ensure your school's data privacy is active before uploading curriculum documents.
* **Privacy Warning:** Do not input student names or private grades. Anonymize data even if using a "Protected" school account.

---

## ✅ How to Know You Have Mastered This Tool
By the end of this guide, you should be able to:
* Switch between Creative, Balanced, and Precise modes effectively based on the task.
* Generate a document and export it directly into Microsoft Word for further editing.
* Use the citations feature to verify three separate facts in a generated lesson plan.
