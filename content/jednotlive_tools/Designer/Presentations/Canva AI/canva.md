# Canva (Magic Studio): Standardized Guide for Teachers

---

## 1. General Use Case
Canva's Magic Studio is an AI-powered suite within the Canva platform designed to simplify the creation of visual educational content. It acts as a co-designer, helping teachers transform text into presentations, generate custom images for lessons, and automate repetitive design tasks.

### 🛠️ Recommended Use Cases
* **Creating Presentations:** Using Magic Design to generate a full presentation deck from a simple text prompt.
* **[Creating Visual Aids](../../../jednotlive_usecases/aktivity_na_hodinu/cvicne_ulohy/practice_questions.md):** Using Magic Media (Text to Image) to create custom illustrations for biology, history, or geography.
* **[Writing Content](../../../jednotlive_usecases/administrativa/pisanie_emailu/writing_email.md):** Using Magic Write to brainstorm lesson ideas or draft educational posts directly within a design.

---

## 🔄 The Step-by-Step UI Workflow

`[Access/Login] ➔ [Prompting/Input] ➔ [Interacting/Refining] ➔ [Finalizing/Exporting]`

### Step 1: Accessing the Tool
Visit canva.com and log in with your account. Teachers should ideally use a "Canva for Education" account for free access to premium features.

### Step 2: Providing Input
In the search bar or within a blank design, look for the "Magic Studio" icon or use the "/" command to trigger "Magic Write." For images, go to the "Elements" tab and select "Magic Media."

### Step 3: Refining the Result
Use the "Magic Switch" tool to change the format (e.g., from a Doc to a Presentation). Use the "Edit" menu to adjust AI-generated images or the "Rewrite" feature in Magic Write to change the tone.

## ✍️ Effective Prompting for Canva

Canva's creative models respond best to descriptive keywords and style references.

### How to Write Prompts for this Tool:
* **Be Specific about Style:** Instead of "A cat," use "A 3D Pixar-style cat sitting on a school desk, vibrant colors."
* **Define the Audience:** For Magic Write, specify: "Explain photosynthesis for a 10-year-old in 3 bullet points."
* **Compositional Cues:** Use terms like "centered," "minimalist," or "cinematic lighting" for better visual results.

### Example Prompt:
> "Design a 5-slide presentation about Ancient Rome for a 6th-grade history class. Include key topics like the Colosseum, the Senate, and daily life. Style: Educational, engaging, and professional."

---

## ⚠️ Tool-Specific Review Checklist (Shortcomings)

* **Fact-Checking:** Magic Write can occasionally generate incorrect facts or dates. Always verify educational data against a trusted source.
* **Visual Artifacts:** AI-generated images may have distorted limbs or unreadable text. Check all visual details carefully.
* **Layout Overload:** Magic Design can sometimes create cluttered layouts. You may need to manually simplify the final design for clarity.
* **Privacy Warning:** Do not upload student photos or identifiable private data into Canva's AI processing tools.

---

## ✅ How to Know You Have Mastered This Tool
By the end of this guide, you should be able to:
* Successfully generate a full 5-slide presentation from a single text prompt.
* Use Magic Media to create a specific illustration for a lesson handout.
* Use Magic Switch to convert a lesson plan document into a set of presentation slides.
